Limit of a Function, Infinite Limits, and Asymptotes

The limit of a function f(x) is the value that it "approaches" as x "approaches" some value of a.

The Long Evaluation of a Limit

Formally, the limit f(x), as x approaches a, is equal to L, if we can make f(x) abitrarliy close to L by making x sufficiently close to a but not equal to a.
